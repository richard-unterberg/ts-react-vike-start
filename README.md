## Usage

### Install

- clone
- run `npm install` in root

### Development

Start local dev server:
```bash
npm run dev
```

### Build

build production ready copy to `dist` folder:
```bash
npm run build
```

### Preview Build

to preview production-build locally:
```bash
# build before preview
npm run build

# the run preview
npm run preview
```
