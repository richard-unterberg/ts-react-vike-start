ts-react-vike-start

TypeScript + React + Vike

Tailwind CSS + ESLint + Prettier

Featuresome template to provide an additional setup to get TS-React working in Vite with HMR and handful of strict ESLint rules. I want to add to work with properly in my side projects.

Preconditions:

npm or yarn
editorconfigvs code extension
prettiervs code extension
tailwindcssvs code extension

Features:

👏 React + Vike
🎨 Tailwind CSS
✍️ @fontsource
🎛️ Custom Theme Hook
💫 lucide-react icons
🤌 Strict eslint setup
💄 Prettier + Prettier Sort Imports
🚨 Husky + lint-staged
⚙️ editorconfig
🚓 Typescript

Usage

Installation

clone repo and:
run npm install in root

Local Dev

Start local dev server with
npm run dev

Build

build production ready copy to dist folder:
npm run build

Preview

Preview the build from dist folder:
npm run preview
